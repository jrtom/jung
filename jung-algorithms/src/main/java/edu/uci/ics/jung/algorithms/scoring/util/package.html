Utility functions for assigning scores to graph elements; largely for library-internal use. These include:
  • DelegateToEdgeTransformer: delegates a VEPair-to-weight function to a supplied edge-to-weight function
  • NodeScoreTransformer: wrapper around NodeScorer for fetching node scores
  • ScoringUtils: methods for calculating transition probabilities for random-walk-based algorithms
  • UniformDegreeWeight: an edge weight function that assigns weights as uniform transition probabilities to all outgoing edges of a node
  • VEPair: convenience class for associating a node and an edge
