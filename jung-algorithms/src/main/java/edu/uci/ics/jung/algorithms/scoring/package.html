Mechanisms for assigning values (denoting significance, influence, centrality, etc.) to graph elements based on topological properties, including:
  • BarycenterScorer: the sum of the distances to all other nodes
  • BetweennessCentrality: the number of all-pairs shortest paths that the node/edge is on
  • ClosenessCentrality: the mean distance to each other node
  • DegreeScorer: the degree
  • EigenvectorCentrality: the long-term probabilities of random walks passing through the node
  • PageRank: like EigenvectorCentrality, but with a constant probability of the random walk restarting at a uniform-randomly chosen node
  • PageRankWithPriors: like PageRank, but with a user-specified (non-uniform) random restart distribution
  • HITS: hubs-and-authorities (based on complementary random walks)
  • HITSWithPriors: like HITS, but with a user-specified (non-uniform) random restart distribution
  • VoltageScorer: simulated current flow along edges
