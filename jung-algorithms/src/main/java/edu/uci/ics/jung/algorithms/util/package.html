General-purpose (non-graph-specific) algorithmic utilities. These include:
  • IterativeContext and Iterative Process: scaffolding for algorithms that iterate until they converge, or until a maximum number of iterations is reached
  • KMeansClusterer: uses the k-means algorithm to cluster points in d-dimensional space into k clusters
  • MapBinaryHeap: a binary heap implementation that permits efficient element access and update operations
  • SettableTransformer: an extension of Function that allows mutation of the transformation
  • WeightedChoice: an implementation of a non-uniform random item selector with user-specified weights
